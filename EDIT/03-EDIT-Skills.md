# Skills Page (/skills) — Editing Guide

## Skills grid
- What it is: the 4 category cards (Finance & Data, Design & Media, Making & Craft, Culinary), each listing your skills with a level tag (Core / Advanced / Learning).
- Controlled by: `content/skills.yaml`.
- Safe to change:
  - Add, remove, or rename any skill under `items` within a category.
  - Change the `level` of any skill to `"core"`, `"advanced"`, or `"learning"` — this changes the little badge/color shown next to it.
  - Rename a category's `title` if you want (e.g. "Making & Craft" → something else).
- Leave alone: the `id` and `icon` fields — these control which icon shows up and shouldn't be changed unless you also know which icon names are available.
